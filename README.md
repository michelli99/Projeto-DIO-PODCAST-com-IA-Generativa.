# Projeto-DIO-PODCAST-com-IA-Generativa.
Este é um Projeto de PODCAST, Insticado pelo BootCamp da Dio, em  parceria com a Universia e Santander sobre IA Generativas.

IAs Utilizadas:

ChatGPT - Routeirização;

IA do Canva - Gerar imagens;

MidJourney - Gerar imagens;

NotebookLM - Gerar texto e áudios.
